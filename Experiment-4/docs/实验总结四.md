# 二、实验原理
## 2.1 图像分割任务概述
图像分割（Semantic Segmentation）是计算机视觉中的基础任务，目标是对图像中每个像素进行分类，将其分配到预定义的语义类别中。与目标检测输出边界框不同，分割输出与原图等尺寸的像素级标签图（Label Map）。主要方法包括基于FCN的全卷积方法、基于编解码器结构的方法（如U-Net、SegNet）、以及引入空洞卷积和条件随机场的方法（如DeepLab系列）。
## 2.2 FCN（全卷积网络）
FCN由Long等（2015）提出，是将CNN应用于语义分割的开创性工作。其核心思想是：
(1) 全卷积化：将传统CNN中的全连接层替换为卷积层，使网络可以接受任意尺寸的输入；
(2) 上采样（Upsampling）：使用转置卷积（反卷积）将低分辨率的特征图恢复到原始图像尺寸；
(3) 跳跃连接（Skip Connection）：融合深层的语义信息和浅层的空间细节信息，FCN-8s融合了第3、4、5层的特征，比FCN-32s（仅使用第5层）具有更精细的分割边界。
FCN-ResNet50使用ResNet-50作为骨干网络，通过空洞卷积保持特征图分辨率，最终通过双线性插值上采样到输入尺寸。
## 2.3 DeepLabV3
DeepLabV3由Chen等（2017）提出，在FCN基础上引入了多项改进：
(1) 空洞空间金字塔池化（ASPP）：使用不同膨胀率（dilation rate）的空洞卷积并行提取多尺度特征，捕捉不同大小的目标。同时加入全局平均池化分支获取全局上下文信息；
(2) 空洞卷积（Atrous/Dilated Convolution）：在不降低分辨率的情况下扩大感受野，膨胀率r=1为标准卷积，r=2时隔一个像素采样，有效感受野扩大但参数量不变；
(3) 改进的骨干网络：使用ResNet-50并在最后两个block中使用空洞卷积，保持高分辨率特征图；
(4) 条件随机场（CRF）后处理（DeepLabV1/V2使用，V3中移除）：利用像素间的颜色相似性和空间接近性细化分割边界。
## 2.4 上采样方法
语义分割需要将低分辨率特征图恢复到输入尺寸，常用的上采样方法包括：
(1) 转置卷积（Transposed Convolution）：可学习的上采样，通过反卷积操作恢复空间分辨率；
(2) 双线性插值（Bilinear Interpolation）：固定权重的上采样，计算量小但不可学习；
(3) 像素洗牌（Pixel Shuffle）：通过通道到空间的变换实现上采样。

## 5.4 结构差异分析
FCN与DeepLabV3的核心结构差异如下：
(1) 多尺度处理：FCN通过跳跃连接融合多尺度特征，DeepLabV3通过ASPP并行提取多尺度特征，后者在处理不同大小目标时更加灵活；
(2) 感受野：DeepLabV3使用空洞卷积扩大感受野，不增加参数量也不降低分辨率，能捕捉更大范围的上下文信息；
(3) 全局信息：DeepLabV3的ASPP包含全局平均池化分支，显式引入全局上下文信息，有助于理解图像整体语义；
(4) 边界质量：DeepLabV3通常产生更平滑、更精确的分割边界，这得益于其更强大的特征提取能力。
# 六、总结与心得体会
通过本次图像分割实验，我深入理解了FCN和DeepLabV3两种典型分割网络的设计理念，主要收获如下：
1. 全卷积思想：FCN将传统分类网络改造为像素级预测网络的核心思想——去除全连接层、使用转置卷积上采样——为后续所有分割网络奠定了基础。
2. 多尺度特征的重要性：无论是FCN的跳跃连接还是DeepLabV3的ASPP，都强调了多尺度特征融合对分割精度的关键作用。深层特征提供语义信息，浅层特征提供空间细节。
3. 空洞卷积的巧妙设计：在不增加参数量的情况下扩大感受野，是解决"分辨率 vs 感受野"矛盾的有效方法。
4. 分割与检测的关系：分割提供了比检测更精细的像素级信息，在自动驾驶（道路/行人分割）、医学影像（器官/病灶分割）等领域有广泛应用。
5. 预训练模型的价值：通过迁移学习，预训练的分割模型可以快速应用于新场景，大大降低了对标注数据量的需求。

def colorize_seg(seg, palette):
    h, w = seg.shape
    color_img = np.zeros((h, w, 3), dtype=np.uint8)
    for c in range(len(palette)):
        mask = seg == c
        color_img[mask] = palette[c]
    return color_img

def overlay_seg(img, seg, palette, alpha=0.5):
    img_np = np.array(img)
    color_seg = colorize_seg(seg, palette)
    h, w = min(img_np.shape[0], color_seg.shape[0]), min(img_np.shape[1], color_seg.shape[1])
    blended = img_np[:h, :w].copy()
    mask = color_seg[:h, :w].sum(axis=2) > 0
    blended[mask] = (alpha * color_seg[:h, :w][mask] + (1-alpha) * img_np[:h, :w][mask]).astype(np.uint8)
    return blended
